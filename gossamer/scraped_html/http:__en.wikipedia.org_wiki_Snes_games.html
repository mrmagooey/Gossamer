This is a list of 784 games for the Super Nintendo Entertainment System (721 in NA, 533 in EU and 470 in both), organized alphabetically by name. This list is split between two articles. This article encompasses all games starting with the letters A-M, and numbers 0-9. For the list of games starting with the letters N-Z, see List of Super Nintendo Entertainment System games (N-Z).

This is intended as a complete list of all officially released SNES games for all SNES markets. Any games not officially licensed by Nintendo are marked (unl). Items specific to North America are marked 'NA'. Items specific to a European release show 'EU'. Releases specific to other countries will use an appropriate abbreviation. Where known, all available SNES regions are listed with appropriate flags in the Region column. The release date, developer, and publisher columns should show the best known information about the earliest release for Super Nintendo, with any significant differences between releases noted. For additional release information on a given game, see that game's article. All known alternate SNES game titles are listed.

For a similar list pertaining to the Super Famicom in Japan, see: List of Super Famicom games. All games fan-translated from Japanese that were never released in North America and/or Europe will be found there.

For a list by Genre classification, please see: List of Super Famicom and Super Nintendo games by genre

See Lists of video games for related lists.
